Build images inside minikube (example):
- minikube image build -t gameforum-backend:latest -f forum/Dockerfile forum
- minikube image build -t gameforum-frontend:latest -f forum/src/frontend/GameForum-master/Dockerfile forum/src/frontend/GameForum-master

Apply manifests:
- kubectl apply -f k8s/

Access:
- Ingress: map gameforum.local to minikube ip, then open http://gameforum.local
- NodePort: http://<minikube-ip>:30080 (frontend) and http://<minikube-ip>:30081 (backend)
